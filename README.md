# 🌍 InAmigos Foundation - NGO Awareness Website

A simple and responsive NGO awareness website developed using **HTML** and **CSS** as part of an internship task.

## 📌 About the Project

This website spreads awareness about **InAmigos Foundation** and its initiatives. It showcases the NGO's mission, ongoing projects, social impact, and encourages people to volunteer and support the organization.

## ✨ Features

- 🏠 Responsive Home Page
- 📖 About the NGO
- 📸 Hero Banner
- 💚 Ongoing Projects
  - Project SEVA
  - Bachpanshala
  - Project UDAAN
  - Project JEEV
  - Project PRAKRITI
  - Project VIKAS
- 📊 Social Impact Section
- 🖼️ Image Gallery
- 🤝 Call-to-Action Section
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🎯 Objective

To create an informative and visually appealing webpage that raises awareness about the initiatives and social impact of InAmigos Foundation.

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
